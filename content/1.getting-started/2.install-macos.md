---
title: "Install on macOS"
description: "Install ZisK on macOS 14+ (Sonoma or later) for both Intel and Apple Silicon."
---

macOS works well for developing and testing ZisK programs. You can write guest code, run it in the emulator, and generate proofs, all on your Mac. Both Intel and Apple Silicon are supported (macOS 14 Sonoma or later).

The main limitation is that the fast ASM backend and GPU acceleration are Linux-only. On macOS, you'll use the emulator backend (`.emu()`) for execution and proving.

## Dependencies

Install [Homebrew](https://brew.sh/) and [Xcode](https://developer.apple.com/xcode/) first.

```bash
brew reinstall jq curl libomp protobuf openssl nasm \
  pkgconf open-mpi libffi nlohmann-json libsodium riscv-tools
```

You also need [Rust](https://www.rust-lang.org/tools/install) and [Git](https://git-scm.com/).

---

## Install ZisK

```bash
curl https://raw.githubusercontent.com/0xPolygonHermez/zisk/main/ziskup/install.sh | bash
```

Choose your setup option:

| Option | What it installs | Use case |
| --- | --- | --- |
| **Proving key** (default) | Full proving key | Generate and verify proofs |
| **Proving key (no constant tree files)** | Proving key without constant trees | Proving key without pre-generated constant trees |
| **Verify key** | Verification key only | Verify proofs only |
| **None** | Toolchain only | Compile and emulate programs |

---

## Verify

```bash
rustup toolchain list
```

You should see `zisk` in the list.

```bash
cargo-zisk --version
```

---

## Generate constant trees

If you installed the proving key:

```bash
cargo-zisk check-setup -a
```

---

## The `disable_distributed` feature

When using the ZisK SDK on macOS, you **must** enable the `disable_distributed` feature since OpenMPI is not fully supported:

```toml
[dependencies]
zisk-sdk = { git = "https://github.com/0xPolygonHermez/zisk.git", features = ["disable_distributed"] }
```

Without this, your host program will fail to compile.

## Updating

```bash
ziskup
```

## Uninstall

```bash
rustup uninstall zisk
rm -rf $HOME/.zisk
```

Once installed, head to [Quickstart](/getting-started/getting-started-with-zisk) to create your first project and generate a proof.
